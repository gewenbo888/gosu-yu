# Gosu Yu (戈苏御)

Character IP showcase exploring the five facets of Gosu Yu.

[![Live Demo](https://img.shields.io/badge/demo-gosu--yu.vercel.app-gold)](https://gosu-yu.vercel.app)

## Overview

A bilingual (Chinese / English) single-page site presenting the character IP "Gosu Yu" through five archetypal facets:

- **戈 (Warrior / Craftsman)** -- discipline and precision
- **苏 (Visionary Thinker)** -- imagination and insight
- **御 (Charismatic Leader)** -- command and presence
- **风 (Witty Trickster)** -- humor and adaptability
- **Five-Elements Synthesis** -- how the facets interrelate

Features an animated Chinese character hero, five-elements orbiting canvas, trait progress bars, and an origin timeline.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, HTML5 Canvas
- **Aesthetic:** Dark ink-wash (shuimo) theme with gold and vermillion accents
- **Typography:** Noto Serif SC (Chinese) + Cormorant Garamond (English)
- **Language:** Bilingual CN / EN throughout
- **Deployment:** Vercel
